
The attached code has the following opeartions:

1. baseline_model.ipynb - To develop baseline DNN model for Transfer Learning.
2. Indoor_env_TL.ipynb - For transferring baseline model to each new wireless environment.
3. cnn_similarity_measure.ipynb - To develop a data-driven similarity measure.
4. accuracy.ipynb - To calculate different performance measures. 
5. no_room_baseline.h5 - Saved baseline DNN model.
